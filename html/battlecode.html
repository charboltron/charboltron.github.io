MIT Battlecode Software Engineering

In Feb-March 2020, I participated in the MIT Battlecode challenge. Although our team did not make it to the finals, the project is mostly about applied software engineering in a multithreaded environment, with an emphasis on developing skills to be a successful project manager and managing a team. I was manager for a team of three students, and through this project I further honed my Java skills, learned about Travis CI and integerating several other github tools for code review and inspection.
