  • ChatBox: An IRC Chatroom

    • Created May 2019, this IRC chatroom, affectionately dubbed "the ChatBox" was a cool experiment with Python's Asynchronous I/O package asyncio. This partner project was a collaboration with my friend and fellow student Evan DePosit, and it was completed as our final programming project for our Internet and Networking Protocols course. The IRC Chatroom establishes a TCP connection with a local server and allows multiple connections, room creation, joining, leaving, message broadcasting, and several other interesting features. It also includes some fun techno-spacey sound effects generated by yours truly using Python's wave package along with a little tinkering in Logic Pro X. In the picture, the server log appears on the left, the chatting clients on the right.
