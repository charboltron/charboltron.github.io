Trivia Night: A Fullstack Application

This project was completed by a team of three students. Trivia Night is a fullstack application which uses API calls to the Open Trivia Database for trivia questions and presents them to the user in a game format. We worked on this application for five weeks. The final product had several features. The user could play a quick game, which was just a random API call with 10 random questions. If the user signed up, then they could select a specified number of questions based on category and difficulty. User sign up was implemented using a postgreSQL database (this was not implemented professionally, but was a great learning opportunity). This was a really rewarding project to work on and a great exercise in being part of a small team with a lot of independence. Written in Javascript with HTML/CSS and bootstrap.
