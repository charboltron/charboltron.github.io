Project-Silver: A Spreadsheet Parser and RESTful API

This project was completed as a capstone project for Portland State University along with an internship with Portland-based ecommerce and A/B testing consulting company, The Good. For this super-interesting project I worked with a strategist and two software developers over the course of five months (November 2018 - March 2019). The inspiration for this project came from the need to have access to spreadsheet data immediately, in order to perform statistical analysis on the aggregate data. I built a backend spreadsheet data wrangler using Python and Pandas, then built an API to serve the JSONified data using Flask. I also experimented with data plotting in Jupyter and Matplotlib and wrote the API documentation, learning how to use Postman along the way.
