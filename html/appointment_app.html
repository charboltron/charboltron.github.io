Metadate: An Android Application
Created during the summer of 2019, metadate was the final project for an advanced programming in Java course. In addition to being the first Android application I had ever created, this was also the culmination of an 8-week iterative, object-oriented Java program, beginning with a simple Java program, then an HTTP servlet, and finally, a fully-functional small-device application. Built with Android Studio, IntelliJ. Written in Java.
